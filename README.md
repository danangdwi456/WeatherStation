****Urutan Upload yang Benar (Wajib Diingat)****

_**1. Upload File .bin Terlebih Dahulu**_

   Klik Export compiled Binary di Arduino IDE, lalu unggah (timpa) file Main_code_Updated.ino.bin yang ada di repositori GitHub kamu.

_**2. Ubah Angka di version.txt Terakhir**_

  Setelah file .bin dipastikan sudah selesai terunggah, barulah kamu buka file version.txt, ubah angkanya (misalnya dari 1.1 menjadi 1.2), lalu klik Commit changes.
